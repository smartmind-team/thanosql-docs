    시작 전 사전 정보¶

    • 마지막 수정날짜 : 2022-07-14

    1. LIST 구문 개요¶

    사용자는 "LIST" 구문을 사용하여 현재 ThanoSQL의 Pre-built 모델들("THANOSQL MODEL")과 사용자가 만든 모델들("MODEL")을 확인할 수 있습니다.

    2. LIST 구문¶

    "LIST MODEL" 구문은 사용자가 만든 모델들의 리스트를 확인합니다.

    Caution

    단, 저장한 모델이 없는 경우에는 오류가 발생합니다.

    %%thanosql
LIST MODEL

    "LIST THANOSQL MODEL" 구문은 ThanoSQL의 Pre-built 모델들의 리스트를 확인합니다.

    %%thanosql
LIST THANOSQL MODEL
